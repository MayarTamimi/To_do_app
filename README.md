## Features

- login and signup  
- add, edit, and remove any task  
- display the tasks in a table  
- display a task in a grid  
- hunt the ghost game  
- update your name and password  
- add, edit, and remove any status  
- display the page in two languages: English and Arabic

## Technologies Used

- React.js  
- CSS  
- Axios  

## Setup Instructions

- cd Todo  
- npm install  
- npm run dev
